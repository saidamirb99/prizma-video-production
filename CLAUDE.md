# Prizma Video Production

Video production project with full pipeline: scripting, editing, effects, rendering, and motion graphics.

## Structure
- `assets/` — Raw materials (video, audio, images, fonts)
- `projects/` — Individual video project folders
- `exports/` — Final rendered outputs
- `scripts/` — Automation scripts (FFmpeg, Python)
- `templates/` — Reusable motion graphics templates
- `website/` — Prizma website source

## Tools
- **FFmpeg** — Video processing, transcoding, effects
- **fluent-ffmpeg** (Node.js) — Scripted FFmpeg workflows
- **sharp** (Node.js) — Image processing for thumbnails/overlays
- **MoviePy** (Python) — Programmatic video editing
- **Manim** (Python) — Motion graphics and mathematical animations
- **Pillow** (Python) — Image manipulation

## MCP Servers
- **filesystem** — File management within the project
- **playwright** — Browser automation for website building
